# Icon Creation Guide

## Required Icons

You need three PNG files:
- `icon16.png` - 16×16 pixels
- `icon48.png` - 48×48 pixels  
- `icon128.png` - 128×128 pixels

## Quick Options

### Option 1: Use an Icon Generator
1. Create or find a 128×128 icon design
2. Use an online tool to resize:
   - https://www.icoconverter.com/
   - https://www.favicon-generator.org/
   - https://realfavicongenerator.net/

### Option 2: Use Figma/Canva
1. Create a 128×128 canvas
2. Design your icon (shield + lock, or shield + checkmark)
3. Export at all three sizes

### Option 3: Use AI Image Generator
Prompt: "Simple shield icon with lock symbol, security theme, minimal design, transparent background, 128x128 pixels"

### Option 4: Use Free Icon Libraries
- https://www.flaticon.com/ (search "shield security")
- https://icons8.com/
- https://www.iconfinder.com/

## Design Suggestions for AgentTrust

**Theme**: Security, Trust, Protection

**Elements to consider**:
- Shield shape (primary)
- Lock or checkmark inside
- Blue/purple gradient (matches your popup design)
- Minimal, modern style

**Color Palette** (from your popup):
- Primary: #667eea (purple-blue)
- Secondary: #764ba2 (purple)
- Accent: #4caf50 (green for trust)

## Testing

After adding icons:
1. Load extension in Chrome
2. Check toolbar icon (16px)
3. Check chrome://extensions page (48px)
4. Verify all sizes look good
