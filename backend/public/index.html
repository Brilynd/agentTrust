What AgentTrust Does

Five layers of trust between your AI agent and the real world.

Browser Extension

Your command center for AgentTrust. Monitor every agent action in real-time, approve or deny high-risk operations via step-up prompts, manage saved logins and domain permissions, chat with the agent, and create reusable routines. Click the AgentTrust icon in your browser toolbar to open it.

Policy Engine

Every browser action and API call is classified by risk level. High-risk actions require your explicit approval before the agent can proceed.

Audit Trail

Every action is logged to PostgreSQL with timestamps, risk levels, and outcomes. Full history is queryable through the extension's Monitor tab.

Token Vault

OAuth tokens for GitHub, Google, Slack, Microsoft (Outlook, OneDrive, ToDo), and Notion are securely exchanged through Auth0 Token Vault. The agent never sees raw credentials.

Browser Automation

The agent navigates real websites, fills forms, reads content, and manages tabs. Every interaction is observed and verified before and after execution.

Getting Started

Use the browser extension to interact with AgentTrust. The terminal is only needed to start the agent.

1
Open the Extension

Click the AgentTrust icon in your browser toolbar. Sign in if prompted — this is your dashboard for monitoring, chatting, and configuring the agent.

2
Configure Permissions

Go to the Permissions tab to add saved logins, set allowed/blocked domains, high-risk keywords, and step-up approval rules.

3
Give the Agent a Task

Use the Chat tab in the extension to send commands, or type directly in the terminal where the agent is running.

4
Monitor & Approve

Watch live actions in the Monitor tab. When the agent attempts a high-risk action, a step-up approval banner appears — approve or deny it in one click.

Example tasks you can give the agent:

Cross-Service Workflow Research the latest AI regulations, email a summary to legal@company.com, post a TL;DR to #legal-updates on Slack, and schedule a review meeting on Google Calendar
Research & Notify Find the latest news on AI development, send an Outlook email with findings via Microsoft, and create a Notion page with the full report
GitHub & Slack List my open GitHub issues and post a summary to #dev-updates on Slack
Web Navigation Go to amazon.com and find the best-rated wireless headphones under $100
